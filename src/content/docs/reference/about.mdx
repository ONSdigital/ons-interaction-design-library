## What This Library Is

#### The Design Source of Truth
This is the formal Interaction Design Library and the official source for all design decisions, interactions, and component behaviors.

#### A Specification for Handoff
It provides a clear document that explains both how a component looks and exactly how it is intended to work, ensuring a robust process for handoff to Development.

#### Atomic Design Framework
It follows Atomic Design principles, containing the fundamental building blocks (atoms, molecules, organisms) required to create all templates and pages.

#### Finalized and Validated
Everything in this library has undergone multiple rounds of user testing and formal design reviews; it is finalized and agreed.

#### Design Intent & Interaction
It explicitly details behaviors so that no assumptions are made by implementation teams.

## What This Library Is Not

#### A Technical Implementation Guide
This library does not dictate semantic HTML decisions (e.g., `<p>`, `<span>`, or `<h3>` tags). Those are technical front-end decisions owned by Engineering to ensure accessibility and stability.

#### A Repository for Operational Requirements
Unless requirements directly impact the visual design or user behavior, they sit outside this library’s remit.

#### An Implementation Code Library
This documents the design intent; the actual implemented code lives in the Design System. These must remain distinct to avoid a fragmented or disjointed collection of design, SEO, and technical notes.

#### A Creative Review Space
As content is already validated and agreed, this space is for implementation guidance, not for re-litigating design decisions.

## Who This Library Is For

#### Designers
To ensure consistency and a robust process for using pre-validated patterns.

#### Developers
To inspect finalized designs and understand the full "Design Intent" and required behaviors before technical development begins.

#### Stakeholders
To provide transparency and a clear view of the latest, agreed-upon designs for the project.
